================================================================================
                        WAVE 7: SUPERAGENT (Orchestrator)
================================================================================

SUPERAGENT ORCHESTRATOR (superAgent.ts)
  - Intent classification (6 predefined intents)
  - Agent chain routing (1-3 agents per intent)
  - Full workflow orchestration (8 phases)
  - Human-in-loop approval gates
  - Complete audit trail (MongoDB AgentTask)

Intent → Chain Mapping:
  find_leads        → [research, strategy, execution]  (3-agent)
  track_competitors → [research]                        (1-agent)
  generate_campaign → [strategy, execution]             (2-agent)
  weekly_summary    → [strategy]                        (1-agent)
  brand_mentions    → [research]                        (1-agent)
  general_chat      → []                                (0-agent, LLM only)

WORKFLOW PHASES:
  1. Create AgentTask document
  2. Classify intent (LLM → Zod schema)
  3. Fetch business context
  4. Execute agent chain (research → strategy → execution)
  5. User approval gate (optional)
  6. Execution Agent (if approved)
  7. Generate final summary (LLM)
  8. Update AgentTask + return
